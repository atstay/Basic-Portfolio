About me

I'm super stoked to learn to code. I never really did coding aside from editing webpages back in middle school. Unfortunately I lost most that I learned back then, but hopefully it will all come back shortly! So far bootcamp is great! Definitely tough but I'm always up for a challenge. Ideally I would like to eventually work for a cyber security company; however, that of course depends on how well I can pick up basic web development first!
